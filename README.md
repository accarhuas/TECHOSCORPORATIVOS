# Techos Corporativos (Astro + Vercel)

Landing B2B optimizada para SEO de reparacion de techos en Chile.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy en Vercel

1. Importar repositorio en Vercel.
2. Framework preset: `Astro`.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Variable opcional recomendada: `PUBLIC_SITE_URL` con tu dominio final.
